Extension: Regimen Intent (Experimental)

Official URL: http://hl7.org/fhir/us/codex-mopa/StructureDefinition/ocpa-regimen-intent Version: 0.1.0
Standards status: Informative Computable Name: RegimenIntentExtension

The clinical intent of the anti-cancer regimen (e.g., curative, palliative, adjuvant, neoadjuvant, supportive). Applies to both the canonical regimen definition (PlanDefinition) and the patient-specific ordered instance (RequestGroup).

mCODE Migration Candidate — Proposed for inclusion in mCODE STU5 as an extension on anti-cancer regimen profiles.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: PlanDefinition
    • Element ID: RequestGroup

⚠ mCODE Migration Candidate

This Regimen Intent extension is defined in this IG as a temporary home while a formal proposal to incorporate it into mCODE STU5 is developed. It is not intended to be a permanent artifact of the MOPA IG. Once adopted by mCODE, this IG will remove its local definition and reference the mCODE canonical URL instead.

This artifact carries status = draft and experimental = true. Canonical URLs will change at migration. See the mCODE Gap Proposals page for the full proposal backlog.

Usage info

Usages:

  • Use this Extension: Anti-Cancer Regimen PlanDefinition and Anti-Cancer Regimen RequestGroup
  • Examples for this Extension: Bundle/ExampleOrderSelectBundle, Bundle/ExampleOrderSignBundle, DDACTRegimenDefinition, PHDRegimenDefinition... Show 4 more, THRegimenDefinition, RequestGroup/DDACTRegimenOrder, RequestGroup/PHDRegimenOrder and RequestGroup/THRegimenOrder
