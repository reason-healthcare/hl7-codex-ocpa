# OCPA Implementation Guide

Welcome to the **Oncology Prior Authorization (OCPA) Implementation Guide** project.

This repository is now structured as a standard **SUSHI + IG Publisher** FHIR IG project:

- **FSH authoring** lives in `input/fsh/`
- **IG narrative pages** live in `input/pagecontent/`
- **Published guide generation** uses **IG Publisher**
- **Working project documentation** is maintained in the repository-level `docs/` directory

## Getting Started

1. Install **Node.js** and **Java**
2. Run `npm install`
3. Run `npm run build`

## Scope

This IG will capture the formal FHIR artifacts and narrative needed for the OCPA work, while earlier exploratory analysis remains available in the repository documentation.
